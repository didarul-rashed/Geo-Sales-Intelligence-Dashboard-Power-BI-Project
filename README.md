## 🔎 Project Overview

This project originated from a simple business request: **"Can you create a Bangladesh sales map?"**

Instead of delivering a standalone map, I designed a complete **Geo Sales Intelligence Dashboard** that transforms geographical sales data into actionable business insights. The solution combines regional performance monitoring, trend analysis, KPI tracking, and executive-level storytelling into a single interactive platform.

For confidentiality, all visuals, business information, and figures presented in this repository are for demonstration purposes only.

---

## 🎯 Problem Statement

Organizations often have access to large volumes of sales data but struggle to answer critical questions:

- Which regions are driving growth?
- Which territories require immediate attention?
- How is performance trending over time?
- Where should leadership focus resources and strategy?

Traditional reports provided data but lacked context, visibility, and decision-making support.

## 🗂️ Data Model

The dashboard is built on a scalable analytical model connecting:

- Sales Transactions
- Territory Hierarchy
- District Hierarchy
- Area & Region Mapping
- Calendar Dimension
- Performance Metrics

This structure enables seamless drill-down analysis from national performance to individual territory-level insights.
